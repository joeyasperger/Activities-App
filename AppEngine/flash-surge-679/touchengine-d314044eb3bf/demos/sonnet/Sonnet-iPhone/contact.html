Welcome to Sonnet
Sonnet is part of an open source software framework that aims to facilitate communication between the iPhone SDK and Google App engine. It is available here on Google Code.

The text of Wm. Shakespeare's Sonnets can be found online in several places. The version used within this application can be found at this great website.
