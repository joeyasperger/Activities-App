books:
A demo app for both iPhone and GAE that shows all of the features of the library.
    
doctorspagers:

Originally part of gae-json-rest-info, a project to which touchengine owes a dept of gratitude and upon which touchengine was originally based, doctorspagers shows how to autogenerate a model using mogenerator and how to insert the plist generating decorations.  



sonnet:

A proof of concept demo that shows how to grab a plist from a web service from the iPhone, cache that plist for later (offline) use, and show the data therein.  This is the demo application for our IBM Developerworks article available at http://www.ibm.com/developerworks/web/library/wa-aj-iphone/
